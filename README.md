# 某道手游数据库管理工具

## 项目介绍

预览地址

https://wdt.oneddd.site/

## 开发环境配置

Node.js 版本要求：18.17.0+

pnpm 版本要求：7.0.0+

项目启动

```bash
pnpm dev
```

## 环境变量配置

环境变量讲解

- `DB_HOST`：数据库地址
- `DB_PORT`：数据库端口
- `DB_USER`：数据库用户名
- `DB_PASS`：数据库密码

`.env.local` 文件示例

```env
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=123456
```
